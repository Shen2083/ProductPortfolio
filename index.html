EnvisLM: Proprietary Coaching Language Model

AI / ML
S Situation

General-purpose LLMs lack the domain-specific coaching behaviours needed for household financial guidance: they rarely ask reflective questions, fail to reference GBP amounts from transaction context, and provide generic advice rather than concrete, actionable next steps grounded in the user's actual financial data.

T Task

Train a proprietary small language model that outperforms base models on coaching-specific behaviours: reflective questioning, GBP-contextualised responses, concrete actionable guidance, coaching density per word, and multi-turn conversational coherence. Achieve this within a startup budget constraint.

A Action

Generated 28,438 synthetic ChatML coaching dialogues grounded in behavioural finance research. Executed QLoRA fine-tuning of Llama 3 8B on RunPod A100 GPUs. Built a 5-dimension evaluation framework (reflective questions, GBP amounts, concrete next steps, coaching density, multi-turn coherence). Iterated through v0.1 and v0.2 with rigorous eval on 1,330 test examples. Published adapter weights on HuggingFace.

R Result
2x Coaching Content/Word
80.2% ROUGE-L Win Rate
~$23 Total Training Cost
Llama 3 8B QLoRA HuggingFace RunPod A100 ChatML

Envis Insight Engine: Multi-Modal Financial Intelligence

Deep Learning
S Situation

Financial distress detection in existing tools relies on simple threshold rules (e.g., "balance below £100"). This misses the nuanced signals: conversational tone, spending pattern shifts, household relationship dynamics, and temporal context that together indicate a family is approaching financial difficulty.

T Task

Design and train a multi-modal neural network that fuses transaction data, conversational text, and household graph structure to proactively detect distress, frame coaching interventions, assess timing urgency, and identify household tension before it escalates into crisis.

A Action

Architected a 116M-parameter multi-modal network: FinBERT text encoder, 4-layer Transformer for transaction sequences, and 3-layer Graph Attention Network for household relationships. Designed cross-modal gated attention fusion and 6 task heads covering distress detection (with sub-types), coaching framing, timing urgency, delay scoring, household tension, and goal risk assessment.

R Result
0.903 Distress F1 Score
116M Parameters
6 Task Heads
FinBERT Transformer GAT Multi-Modal Fusion PyTorch
